# Get SNP group counts

Tally locus-specific SNP group sizes.

## Usage

``` r
get_SNPgroup_counts(merged_DT, grouping_vars = "Locus", verbose = TRUE)
```

## Arguments

- merged_DT:

  Merged fine-mapping results generated by `echolocatoR::finemap_loci`.

- grouping_vars:

  Column names in `merged_DT` to group counts by.

- verbose:

  Print messages.

## Value

Named list of report tables.
